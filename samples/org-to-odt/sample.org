* 개요

이 문서는 Org-mode에서 ODT/DOC로 변환하는 파이프라인의 샘플입니다.
pandoc 대신 Emacs의 =org-odt-export= 를 사용하여 다음과 같은 이점을 얻습니다.

- =reference.odt= 스타일 정확 적용
- 한글 캡션 (그림, 표)
- 테이블 헤더 배경색 + 테두리 자동 보정
- CSL bibliography (URL 포함 참고문헌)

** 사용법

#+begin_src bash
./run.sh build              # sample.org → odt → doc
./run.sh build resume.org   # 다른 파일도 가능
./run.sh check              # 의존성 확인
#+end_src
