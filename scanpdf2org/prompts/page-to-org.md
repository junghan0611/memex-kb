# scanpdf2org — 페이지 이미지 → org 전사 규칙

이 문서는 스캔된 한글 책 페이지 PNG를 **에이전트(Claude / Codex / Gemini)가
vision으로 직접 읽어 org-mode로 옮길 때** 따르는 SSOT 규칙이다. 전통 OCR
엔진(tesseract 등)은 쓰지 않는다 — 에이전트가 노가다로 정확히 옮긴다.

## 목표

스캔 페이지의 **본문을 의미 손실 없이** 깨끗한 org로 전사한다. 결과물은
두 용도를 동시에 만족한다:

1. **읽기/데이터** — org 지식베이스에서 다룰 본문 (수식은 LaTeX로 보존)
2. **듣기(EPUB/TTS)** — 흐름이 자연스러운 문단 (수식은 어차피 안 읽히므로 OK)

## 절대 원칙

- **의미를 바꾸지 말 것.** 요약·의역·재작성 금지. 있는 글을 그대로 옮긴다.
- **없는 글자를 추측하지 말 것.** 안 보이거나 불확실하면 `[?]`로 표시한다.
- **수식을 임의로 만들지 말 것.** 불확실한 수식은 `[[CHECK: p<쪽>]]`를 남긴다.

## 제거할 것 (페이지 장식)

- 머리말/꼬리말(running header/footer): 예) `장회익`, `1장 · 칸트 철학과 현대 물리학`
- 쪽번호(page number)
- 장식 괘선, 워터마크, 스캔 얼룩

## ⚠️ 독자의 연필 낙서 (중요)

**각 책의 1장은 독자(GLG)가 연필로 끄적여 둔 자국이 있다** — 1장만 읽었기
때문. 지우개로 안 지운 상태로 스캔됨. 따라서:

- 연필 동그라미·밑줄·괄호·화살표·여백 메모는 **원문이 아니다 → 전사하지 않는다.**
- 인쇄된 활자(검은 본문)만 옮긴다. 손글씨(연한 회색/흐린 선)는 무시한다.
- 연필 자국이 활자를 가려 본문이 안 보이면 → `[?]` 또는 `[[CHECK: p<쪽>]]`.
- 예) 본문 옆에 동그라미 친 손글씨 "그림" → 무시. 본문 단어에 친 밑줄 → 단어는 옮기되 밑줄은 무시.

(2장 이후는 낙서가 거의 없을 것 — 1장만 정독했으므로.)

## 줄바꿈/문단 복구

스캔 본문은 물리적 줄바꿈이 많다. **한 문단은 한 줄로 합친다.**

- 잘못된 줄바꿈으로 분리된 문장 → 자연스러운 문단으로 병합
- 글자 사이 과도한 공백(`인 식 의 구 조`)은 정상 띄어쓰기로 정규화
- 단, 진짜 문단 경계(들여쓰기/빈 줄)는 보존

## org 구조 매핑

| 원문 요소 | org |
|---|---|
| 장 (예: 1장 칸트 철학과 현대 물리학) | `* 1장 칸트 철학과 현대 물리학` |
| 절 (예: 1. 인간의 앎은 어떻게 이루어지는가?) | `** 1. 인간의 앎은 어떻게 이루어지는가?` |
| 소절 (예: (1) 보아서 안다는 것) | `*** (1) 보아서 안다는 것` |
| 본문 문단 | 일반 텍스트 (문단 사이 빈 줄) |
| 강조 (드러냄표/볼드) | `*강조*` |
| 인용·따옴 | 원문 그대로 (`'...'`, `"..."`) |

> 페이지 단위로 전사할 때는 그 페이지에 실제로 나타난 헤딩만 단다.
> 페이지 중간부터 시작하면 헤딩 없이 본문부터 이어 쓴다(조립 단계에서 합침).

## 수식

- 인라인 수식: `\( ... \)` (org에서 렌더됨)
- 별행 수식: `\[ ... \]`
- 긴 전개식이라 옮기기 부담되면 → 원문 보존 시도 + 끝에 `[[CHECK: p<쪽>]]`
- 듣기판에서 안 읽히는 건 감수한다 (형님 결정). 수식은 데이터 가치로 org에 남긴다.

## 그림/표

- 그림: `#+caption:` 없이 `[[그림: <간단한 내용 설명>]]` placeholder. 캡션 글자가 있으면 그대로 옮긴다.
- 표: 칸이 단순하면 org 표(`| ... |`)로. 복잡하면 `[[표: <설명>]] [[CHECK: p<쪽>]]`.

## 각주

- 본문 위치에 `[fn:N]` 표시, 페이지 하단 각주 내용은 `[fn:N] ...`로 페이지 끝에 모은다.

## 출력 형식

- **org 본문만** 출력한다. `#+title:` 등 헤더는 조립 단계에서 따로 붙이므로 넣지 않는다.
- 페이지별 전사라면 파일 맨 위에 주석으로 출처를 남긴다:
  `# src: <PDF> 물리 p<NN> (인쇄 p<MM>)`

## 불확실성 마커 요약

| 마커 | 뜻 |
|---|---|
| `[?]` | 글자 판독 불가/불확실 |
| `[[CHECK: p<쪽>]]` | 수식/표 등 사람 검수 필요 |
| `[sic]` | 원문 오탈자를 그대로 옮김 (의도적) |
